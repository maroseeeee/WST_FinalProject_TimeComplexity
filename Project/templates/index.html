Sorting Algorithms

A sorting algorithm is a set of instructions that rearranges a list of elements into a specific order, usually ascending or descending. This order is often numerical or alphabetical, but it can also be based on other criteria, such as date or custom comparisons.
Key Concepts:
Comparison-based sorting: These algorithms compare elements to determine their relative order.

Examples include:

Bubble Sort: Compares adjacent elements and swaps them if they are in the wrong order.

Insertion Sort: Inserts elements into their correct position in a sorted subarray.

Selection Sort: Finds the minimum element in the unsorted part and swaps it with the first unsorted element.

Merge Sort: Divides the list into two halves, recursively sorts each half, and then merges the sorted halves.

Quick Sort: Picks a pivot element, partitions the list into elements smaller and larger than the pivot, and recursively sorts the partitions.
