BUBBLE SORT


Bubble Sort is a simple sorting algorithm that repeatedly steps through the list to be sorted, compares adjacent elements, and swaps them if they are in the wrong order. The process is repeated until no swaps are needed, which means the list is sorted.

COMPLEXITY

Average Complexity O(n2)
Best Case O(n)
Worst Case O(n2)

Pseudocode

BubbleSort(array):
  n = length of array
  for i from 0 to n-1:
    // Flag to detect if any swapping happens
    swapped = false
    for j from 0 to n-i-2:
      if array[j] > array[j+1]:
        // Swap the elements
        temp = array[j]
        array[j] = array[j+1]
        array[j+1] = temp
        swapped = true
    end for
    // If no two elements were swapped, the array is already sorted
    if swapped == false:
      break
  end for
end BubbleSort
