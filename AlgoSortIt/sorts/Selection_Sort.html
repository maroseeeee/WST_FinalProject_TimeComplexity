SELECTION SORT


Selection Sort is a simple, comparison-based sorting algorithm that sorts an array or list by repeatedly finding the minimum (or maximum, depending on order) element from the unsorted portion and moving it to the beginning (or end) of the list.

TIME COMPLEXITY

Average Complexity O(n2)
Best Case O(n2)
Worst Case O(n2) 
Pseudocode
for (j = 0; j < n-1; j++) 
int iMin = j; 
                
for (i = j+1; i < n; i++) 
if (a[i] < a[iMin]) 
iMin = i; 
                
if (iMin != j) 
swap(a[j], a[iMin]);
